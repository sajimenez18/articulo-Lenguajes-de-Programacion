CSS

¿Qué es CSS?

CSS (Cascading Style Sheets) o Hojas de Estilo en Cascada es un lenguaje utilizado para describir la presentación de documentos HTML. Permite separar la estructura del contenido (HTML) de la apariencia visual (colores, tipografía, diseño, etc.), haciendo que el desarrollo web sea más ordenado, eficiente y reutilizable.

¿Para qué sirve CSS?

CSS sirve para dar estilo y formato visual a los elementos HTML de una página web. Con CSS puedes: Cambiar colores, tamaños y tipos de letra. Aplicar márgenes, bordes y rellenos. Posicionar elementos en la página (de forma fija, relativa, absoluta o flexible). Crear efectos visuales como transiciones, animaciones y sombras. Adaptar el diseño a diferentes dispositivos (celulares, tablets, pantallas grandes). Mejorar la usabilidad y experiencia del usuario sin alterar el contenido. En resumen, HTML estructura el contenido y CSS le da vida visual.

Elementos básicos de CSS

Elementos básicos de CSS Selectores: indican qué elementos del documento serán estilizados. Propiedades: determinan qué aspecto del elemento se modificará (como el color o el tamaño de letra). Valores: definen cómo se aplicará cada propiedad. Reglas: combinan selectores, propiedades y valores para aplicar los estilos. Comentarios: permiten agregar anotaciones dentro del código sin afectar su funcionamiento.
